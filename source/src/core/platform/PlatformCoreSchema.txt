Platform Core Schema

Entities:
- User
- Site
- Workspace
- Role
- Permission

Rules:
- every site belongs to a user
- every action must reference userId + siteId
- AI actions must be traceable to a user

Action shape:
Action {
  type: string
  userId: string
  siteId: string
  payload: object
}

Examples:
- SITE_UPDATE
- PAGE_CREATE
- THEME_CHANGE
- AI_GENERATE_LAYOUT

Platform Core is the source of truth. SiteContext is a UI slice of platform state and projects site-specific data into the app.
